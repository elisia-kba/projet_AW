On à implémentés toutes les fonctions de base ainsi que quelques fonctions bonus.
Quand on fait "Entrée" sur une unité, un popup affiche les actions possibles sans avoir à se déplacer.
Si une unité ennemi est proche alors l’option “Attaque” apparaît. Si on se trouve sur une propriété adverse on peut directement la capturer.
Mais si on ne souhaite pas effectuer les actions proposées il suffit de choisir “Rien” afin de pouvoir se déplacer .
On ajoute le choix “Rien” afin de fermer le popup sans avoir cliquer sur “Echap”.
Chaque joueur ne peut capturer qu’une unité à la fois.
